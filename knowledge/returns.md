# Returns and Reverse Logistics Procedures

## Objective
Returns handling protects inventory accuracy and customer recovery while preventing returned goods from contaminating saleable stock.

## Returns Principles
- Every return must have a clear reason and disposition path.
- Returned stock should never go straight into saleable inventory without inspection.
- Reverse logistics should be fast enough to recover value before goods become obsolete or unfit.

## Return Categories
- Customer remorse or commercial return
- Delivery refusal
- Damaged in transit
- Defective product
- Expired or near-expired return
- Recall or regulatory hold

## Standard Returns Process
1. Receive the returned shipment and verify return reference or authorization.
2. Count cartons, units, and visible condition before opening.
3. Move the stock into the dedicated returns zone or quarantine area.
4. Inspect packaging, product condition, expiry, serial, and tamper evidence.
5. Classify the return reason and assign a disposition code.
6. Decide whether the stock is saleable, refurbishable, return-to-vendor, scrap, or pending investigation.
7. Create the appropriate system movement or adjustment with complete notes.
8. Move released stock to the correct location or hold area.

## Disposition Logic
- Return to saleable stock: unopened and compliant product
- Return to quarantine: condition unclear or awaiting quality decision
- Refurbish or rework: product can be restored to usable state
- RTV: supplier should receive the item back
- Scrap or write-off: product has no recoverable value

## Inspection Rules
- Check lot and expiry before any restock decision.
- For electronics or serialized goods, verify serial number matches the original shipment when possible.
- For chilled or frozen returns, verify temperature integrity before accepting as usable.
- Open-box items should follow a stricter re-sale policy than sealed units.

## Fraud and Control Considerations
- Watch for mismatched SKU, wrong serial, or empty-box returns.
- Use photo evidence for damaged returns and customer disputes.
- Separate customer-claim decisions from physical inspection where financial risk is high.

## Exception Handling

### Return Without Reference
1. Move stock to investigation status.
2. Record customer, carrier, and package details.
3. Do not restock until ownership and reason are confirmed.

### Damaged on Return
1. Segregate immediately.
2. Record transit damage versus product defect where possible.
3. Route to claim, RTV, or scrap based on policy.

### Mismatched Item
1. Hold the item in quarantine.
2. Record the mismatch and source details.
3. Escalate potential fraud or customer error.

## Returns KPIs
- Return processing lead time
- Recovery percentage to saleable stock
- RTV cycle time
- Scrap percentage
- Return reason Pareto
- Customer return accuracy

## Consultant Improvement Techniques
- Review return reasons monthly to separate product-quality issues from fulfillment errors.
- Use a standard disposition code list so recovery trends are measurable.
- Move returned saleable stock back to active inventory quickly; slow reverse logistics hides usable inventory.
- Track which customers, carriers, or SKUs create most avoidable returns.
